CryBlend.Omega

Omega CRYENGINE Exporter for Blender

Download as .zip Download as .tar.gz

If you'd like to see the code yourself or get involved in CryBlend in any way, you can view the code on Github. CryBlend is open source!

How to Install CryBlend.Omega

You must obtain a valid installation of Blender 2.68 or 2.69. You can download this from www.blender.org.

This step is optional: To use the pre-packaged installation program you must have some installation of Python 3. Obtain this from the python website: www.python.org. After installing Python you may just run the python script and follow the instructions.

If you do not use the installer, you must copy the folder io_export_cryblend
